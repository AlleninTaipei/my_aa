## 筆記格式

所有筆記均以繁體中文寫成，遵循統一格式慣例：

- 章節用編號標題（`## 1.`、`## 2.`），子節用 `###`
- 比較性說明（A vs B、何時用 X vs Y）轉為表格
- 程式碼區塊附語言標記
- 章節之間加 `---` 水平線

部分筆記來源為 XML RAG 格式或 Skilljar HTML 課程匯出，已完整重寫為 Markdown，並移除課程包裝內容（Key Takeaways、Lesson Reflection）。
